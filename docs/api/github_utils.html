Utilidades de GitHub

Módulo de utilidades para interacción con la API de GitHub.

Este módulo es parte de Code Empathizer, una herramienta para medir la alineación entre el código de una empresa y sus candidatos.

class github_utils.GitHubRepo[fuente]

Bases: object

Cliente para interactuar con repositorios de GitHub.

Esta clase proporciona métodos para analizar repositorios de GitHub, extrayendo código fuente y calculando métricas de calidad mediante los analizadores de lenguaje disponibles.

Gestiona automáticamente los límites de la API de GitHub y proporciona reintentos cuando se alcanzan los límites de velocidad.
