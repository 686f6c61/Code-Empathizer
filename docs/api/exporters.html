Exporters

Módulo de exportación de resultados.

Este módulo es parte de Code Empathizer, una herramienta para medir la alineación entre el código de una empresa y sus candidatos.

Autor: https://github.com/686f6c61 Repositorio: https://github.com/686f6c61/Code-Empathizer Fecha: Noviembre 2025 Licencia: MIT

class exporters.Exporter[fuente]

Bases: object

Gestor de exportación de resultados en múltiples formatos.

Soporta exportación a texto plano, JSON y HTML con plantillas personalizadas para visualización interactiva.

format_date(value)[fuente]

Filtro Jinja2 para formatear fechas.

Parámetros:

value (Any) – Fecha en formato ISO string o datetime object.

Devuelve:

Fecha formateada como DD/MM/YYYY HH:MM:SS.

Tipo del valor devuelto:

str

exportar_txt(metricas, timestamp)[fuente]

Exporta los resultados a archivo de texto plano.

Genera un informe detallado con formato de tabla que incluye: - Información de repositorios - Puntuación de empatía - Métricas por categoría - Recomendaciones

Parámetros:

  • metricas (Dict[str, Any]) – Diccionario con los resultados del análisis.

  • timestamp (str) – Marca de tiempo para el nombre del archivo.

Muestra:

IOError – Si no se puede escribir el archivo.

Tipo del valor devuelto:

None

exportar_json(metricas, timestamp)[fuente]

Exporta los resultados a formato JSON.

Útil para procesamiento posterior o integración con otras herramientas.

Parámetros:

  • metricas (Dict[str, Any]) – Diccionario con los resultados del análisis.

  • timestamp (str) – Marca de tiempo para el nombre del archivo.

Muestra:

IOError – Si no se puede escribir el archivo.

Tipo del valor devuelto:

None

exportar_html(metricas, timestamp, dashboard=False)[fuente]

Exporta los resultados a formato HTML.

Puede generar un reporte estático o un dashboard interactivo con gráficos usando Chart.js.

Parámetros:

  • metricas (Dict[str, Any]) – Diccionario con los resultados del análisis.

  • timestamp (str) – Marca de tiempo para el nombre del archivo.

  • dashboard (bool) – Si True, genera dashboard interactivo.

Muestra:

  • IOError – Si no se puede escribir el archivo.

  • TemplateNotFound – Si no encuentra las plantillas HTML.

Tipo del valor devuelto:

None

exportar_equipo(resultados_equipo, timestamp)[fuente]

Genera un reporte especial para análisis de equipo.

Tipo del valor devuelto:

None

exportar_graficas_zip(metricas, timestamp)[fuente]

Exporta todas las gráficas generadas en un archivo ZIP.

Genera archivos HTML individuales para cada gráfica (radar, barras, comparación por categorías) y los empaqueta en un ZIP para fácil descarga y distribución.

Parámetros:

  • metricas (Dict[str, Any]) – Diccionario con los resultados del análisis.

  • timestamp (str) – Marca de tiempo para el nombre del archivo.

Devuelve:

Ruta al archivo ZIP generado.

Tipo del valor devuelto:

str

Muestra:

IOError – Si no se puede escribir el archivo ZIP.
